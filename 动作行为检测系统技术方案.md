# 动作行为检测系统技术方案

## 📐 系统逻辑

```
摄像头视频流
     │
     ▼
┌─────────────────┐
│ YOLOv8-Pose    │ ──→ 人体骨骼关键点 (17点)
└─────────────────┘
     │
     ▼
┌─────────────────┐
│ STGCN 行为识别  │ ──→ 动作分类 (坐/站/走/摔倒...)
└─────────────────┘
     │
     ▼
┌─────────────────┐
│  行为统计模块    │ ──→ 久坐时长、活动统计、动作变化
└─────────────────┘
     │
     ▼
┌─────────────────┐
│  健康日志       │ ──→ SQLite 持久化存储
└─────────────────┘
     │
     ▼ 定时触发 (可配置)
┌─────────────────┐
│  LLM 分析模块   │ ──→ 健康分析 + 建议
└─────────────────┘
```

---

## 💼 工作内容

### 1. 姿态估计
- 使用 YOLOv8-Pose 检测人体骨骼 17 个关键点（COCO 格式）
- 输出：关键点坐标 (x, y) + 置信度
- 技术：YOLOv8-Pose、OpenCV

### 2. 行为识别
- STGCN 时空图卷积网络进行动作分类
- 输入：连续 30 帧关键点序列
- 输出：动作类别（坐、站、走、跑、摔倒、弯腰、举手等）
- 技术：STGCN (PyTorch)、数据归一化、平滑滤波

### 3. 行为统计
- 统计各动作持续时长（久坐检测）
- 统计每日活动量、动作变化次数
- 异常行为检测（摔倒自动报警）
- 技术：NumPy 数据处理

### 4. 健康日志存储
- 记录到 SQLite 数据库
- 字段：时间戳、动作、置信度、持续时长
- 支持日报/周报查询统计
- 技术：SQLite、Python

### 5. LLM 定时分析
- 定时任务触发（可自由配置时间）
- 读取健康日志数据
- LLM 分析健康状况，生成建议
- 技术：APScheduler、LangChain、GPT-4

### 6. 部署优化
- 模型导出 ONNX
- TensorRT 加速推理
- 技术：ONNX Runtime、TensorRT
